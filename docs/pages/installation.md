# Installation

## Prerequisites

- Modern web browser (Chrome, Firefox, Safari, or Edge)
- Node.js 16+ (for development)
- npm or yarn

## Installation

### Using npm

```bash
npm install qucat-circuit-generator
```

### From Source

```bash
git clone https://github.com/jurra/qucat-circuit-generator.git
cd qucat-circuit-generator
npm install
npm run build
```

## Running the Application

### Development Mode

```bash
npm run serve
```

Then open `http://localhost:8080` in your browser.

### Production Build

```bash
npm run build
```

The built application will be in the `dist/` directory.

## Standalone HTML

You can also use the pre-built HTML file directly:

1. Download `jscircuit.html` from the releases page
2. Open it in any modern web browser
3. Start designing circuits!
